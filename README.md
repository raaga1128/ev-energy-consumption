*****EV Highway Energy Consumption Prediction Dashboard 🚗⚡*****

An intelligent EV Highway Energy Consumption Prediction System built using HTML, CSS, JavaScript, ESP32 IoT simulation, and ML-inspired analytics to estimate electric vehicle battery usage, charging requirements, and highway driving efficiency in real time.

This project combines:

EV energy prediction
IoT sensor monitoring
Physics + ML ensemble modeling
Live traffic simulation
Battery SOC estimation
Highway EV route analysis

Designed as a research prototype for smart EV charging infrastructure and highway energy optimization.

*Built using:*
ESP32
MPU6050
DHT11
HTML, CSS, JavaScript
Chart.js

****Machine Learning Inspired Prediction Engine****

***Physics-informed ensemble model using:***
1. Aerodynamic drag estimation
2. Rolling resistance
3. Traffic stop-go penalty
4. Li-ion thermal efficiency
5. SOC efficiency modeling

***Outputs:***
1. Prediction confidence interval
2. Efficiency metrics
3. Feature contribution analysis

***Interactive Dashboard***
Includes:
1. Real-time charts
2. Battery SOC gauge
3. Energy factor breakdown
4. Charging station recommendations
5. Trip segment logging
